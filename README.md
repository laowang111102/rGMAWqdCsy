# 美容院管理系统简介 java1020

## 项目概述

美容院管理系统是基于Python+Vue开发的一个前后端分离的项目，主要用于大学生课程设计作业。该系统有助于学习并掌握Python编程技能，同时提升项目设计与开发能力。

## 技术栈

- **后端**：Python 3.8 + Django 3.2
- **前端**：JavaScript + Vue
- **数据库**：MySQL 5.7
- **开发平台**：PyCharm + VSCode
- **运行环境**：Windows 10/11

## 功能模块

- **服务管理**：录入、修改、查询服务基本信息，如名称、简介等。
- **类型管理**：管理服务类型信息，包括类型名称。
- **订单管理**：管理所有订单信息。
- **美容师管理**：管理美容师信息。
- **用户管理**：管理用户信息，包括新增、编辑和删除。
- **轮播图管理**：发布轮播图，供全站用户浏览。
- **评论管理**：用户发布评论，管理员维护评论。
- **注册登录**：用户通过注册登录使用系统。
- **门户浏览**：用户浏览服务列表，包括最新和最热。
- **热门推荐**：基于点击率排序算法进行热门推荐。
- **用户中心**：修改用户资料，管理用户信息、密码、收藏等。
- **我的预约**：查看购买服务的预约信息。
- **意见反馈**：用户提交意见反馈。
- **模糊搜索**：支持模糊搜索服务信息。
- **服务评论**：用户在详情页下评论服务。

## 系统角色

- **服务管理员**：负责服务信息、轮播图和评论的管理。
- **用户**：注册、登录、使用系统功能，如预约、评论等。
